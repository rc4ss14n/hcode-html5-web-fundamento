A tag link aceita todos os atributos globais. E possui atributos específicos listados abaixo.

Atributos CSS:

  • Display = none;

Atributos específicos da tag link.

  • crossorigin - anonymous, use-credentials - Como o elemento lida com solicitações de origem cruzada.
  • href - URL - Destino do link
  • hreflang - código da linguagem
  • media - Em qual disposito o documento se aplica.
  • rel - alternate, author, dns-prefetch, help, icon, licence, next, pingback, preconnect, prefetch, preload, prerender, prev, search, stylesheet - Define o tipo de relacionamento entre os links.
  • sizes - altura X largura
  • type - tipo de midia ao documento associado.
