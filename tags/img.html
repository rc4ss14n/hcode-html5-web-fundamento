Sobre a tag img.

Tag usada para inserir imagens na página.
A tag não possui conteúdo.
Os atributos src e alt são obrigatórios.

A tag img possui os atributos globais e os atributos expecíficos abaixo.

Atributos particulares.

  • alt - Definir um texto alternativo para a imagem - texto
  • crossorigin - Defini como o atributo lida com solicitações de origem cruzada - anonumous,use-credentials.
  • height - definir a altura da imagem. - número.
  • ismap - Defini a imagem como um mapa de imagens do lado do servidor. - ismap
  • longdesc - Define uam descrição detalhada da imagem. - #IDDaDescrição
  • sizes - Definir os tamanhos alternativos da imagem.
  • src - Definir a fonte/link da imagem. - URL.
  • srcset - Definir a URL da imagem em diferentes situações. - URl.
  • usemap - Definir a imagem como um mapa de imagem do lado do servidor. - #nomeDoMapa
  • width - Definir a largura da imagem. - número.

O estilo CSS padrão é:

  • img { display: inline-block; }
