Tag embed é utilizada para embutir objetos à página HTML.
Ela herda todos os atributos globais e possui os atributos expecíficos abaixo.

  • height - pixel - definir a altura
  • src - URL - aminho do objeto
  • type - tipo de conteúdo embutido
  • width - pixel - largura

Estilo CSS padrão é :

  • embed:focus { outline: none; }
