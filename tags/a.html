Sobre a tag a.

Tag usada para criação de links e vincular uma página com outra.
Por padrão o conteúdo é mostrado sublinhado.

A tag a possui os atributos globais e os atributos expecíficos abaixo.

Atributos particulares.

  • download - Força o download do link destino - Se definido um texto no valor, o texto será o nome do download.
  • href - Define qual o endereço do link destino. - URL
  • hreflang - Definir a linguagem do link destino - Código da linguagem.
  • media - Definir a midia ou dispositivo do link destino. - Media query.
  • ping - Endereço de rastreio para monitorar cliques no link destino. - URL.
  • referrepolicy - Definir qual o referenciado enviar. - no-referrer, no-referrer-when-downgrade, origin, origin-when-cross-origin, unsafe-url
  • rel - Definir a relação entre o cod atual e o destino. - alternate, author, bookmark, external, help, license, next, nofollow, norerrer, noopener, prev, search, tag.
  • target - Definir onde o link será aberto. - _blank, _parent, _self, _top, [nome_do_frame].
  • type - Definir o tipo de midia no link destino. - Tipo de midia (MINE type).

O estilo CSS padrão é:

  • a:link { color:blue; text-decoration: underline; cursor: pointer; }
  • a:visited { color: #551A8B; }
  • a:link:active { color: red; }
