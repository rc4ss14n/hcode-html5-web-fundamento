A tag object possui os atributos globais e possui os atributos expecíficos abaixo.

  • data - URL - caminho
  • form - ID de formulário
  • height - altura
  • name - nome do objeto
  • type - tipo de objeto
  • usemap - mapa de objeto
  • width - largura

O estilo de CSS padrão e:

  • object:focus { outline: none; }
