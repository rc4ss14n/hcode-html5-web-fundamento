Sobre a tag figure.

Tag usada para definir uma foto, ilustração, diagrama, entre outras coisas.

A tag figure possui os atributos globais e não possui atributos expecíficos.

O estilo CSS padrão é:


  • figure {
    display: block;
    margin-top: 1rem;
    margin-butto: 1rem;
    margin-left: 40px;
    margin-right: 40px;
    }
