Sobre a tag audio.

Tag usada para definir um som, música ou transmissão de áudio.

Formatos de áudio suportados.

  • .mp3
  • .wav
  • .ogg

Qualquer texto que estiver dentro da tag, será exibido em navegadores que não suportam a tag audio.

A tag audio possui os atributos globais e possui os atributos expecíficos abixo.

  • autoplay - Começa a reproduzir assim que os dados do áudio estiverem disponíveis - autoplay
  • controls - Habilita os controles de reprodução de áudio. - controls
  • loop - Define se o áudio deve começar e ficar repetindo quando terminar - loop
  • muted - Define o áudio como mudo - muted
  • preload - Defini como o áudio deve ser carregado quando a página é carregada - auto, metadata, none
  • src - Definir qual o caminho da fonte/link do áudio - URL

A tag audio não possui nenhum estilo CSS.
