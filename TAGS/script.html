Utilizada para criar script dentro da página ou importar arquivos de script externos, utilizando como padrão a linguagem Java Script como base.

A tag SCRIPT aceita os atributos globais e tem atributos exclusívos.

  • async - executa script externo assincronamente = async
  • charset - define o padrão de caracteres usado, exemplo 'utf-8'
  • defer - será executado depois de carregada a página
  • src - importar scripts = URL
  • type - tipo de midia

Estilos de CSS

  • display = none
